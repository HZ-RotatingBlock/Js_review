# NodeJS/MongoDB简易博客

## 功能模块

### 前端
* 首页
* 列表页 
	* 分类列表
* 内容页 
	* 评论
* 注册
* 登录

### 后端

* 登录
* 分类管理
	* 分类列表
	* 添加分类
	* 修改分类
	* 删除分类
* 内容管理
	* 内容列表
	* 添加内容
	* 修改内容
	* 删除内容
* 用户列表
## 技术构成

* 服务端 -Node.js/Express4
* 模板引擎 -swig
* 数据库 -MongoDB
* 部署（阿里云服务器 or DigitalOcean）Centos7.2系统/nginx(备案中)
